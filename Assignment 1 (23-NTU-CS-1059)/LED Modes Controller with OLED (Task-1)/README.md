## Pin Diagram
| Component    | Color       | ESP32 Pin |
| :----------- | :---------- | :-------: |
| LED1         | Yellow      |    16     |
| LED2         | Blue        |    18     |
| LED3         | Cyan        |    17     |
| Mode Button  | Red Button |    33     |
| Reset Button | Grey Button  |    27     |
| OLED         | SDA         |    21     |
| OLED         | SCL         |    22     |
